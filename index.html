Шаг 1. Установка

  • ТОЛЬКО iOS: Установите напрямую из App Store.
  • macOS, Android, Windows: Скачайте актуальную версию со страницы релизов GitHub - ссылка
    • Android: файл android_arm.apk
    • Windows: файл windows_x64.exe
    • macOS: файл macos_universal.dmg (НЕТ, на macOS нельзя поставить из AppStore, там устаревшая версия почему-то)

Шаг 2. Первый запуск

При открытии приложения нажимайте «Next», пока не дойдете до экрана «Add Profile»:

  • Нажмите Accept & Continue.
  • Выберите язык: English.
  • Выберите регион: Russian Federation (если не определился автоматически).
  • Экран Preset [Custom diversion rules]: ничего не меняйте.
  • Параметр Novice mode: оставьте включенным.
  • Нажмите Done.

Шаг 3. Импорт настроек

Вы оказались на экране «Add Profile». Если нет, то на главном экране нажмите Add Profile.

  • Выберите пункт Backup and Sync
  • Выберите Import and Export
  • Нажмите Import from URL
  • Вставьте следующую ссылку:
    https://github.com/noideastowork/vpn-api/raw/refs/heads/main/FAQ/Karing_1.2.3.857_ios_2026-03-16-1246.backup.zip
  • В появившемся окне нажмите OK.
  • Если при импорте произошла ошибка, значит у вас частично заблокирован GitHub. В этом случае скачайте вручную файл - Открыть Google Drive с файлом. А потом вернитесь назад и нажмите Import (НЕ Import from URL) и выберете скачанный файл Karing_1.2.3.857...backup (1).zip
  • Важно: Полностью перезапустите приложение (или перезагрузите устройство), чтобы настройки вступили в силу.

Шаг 4. Добавление подписок

  • Скопируйте первую ссылку-подписку из этого приложения.
  • В Karing нажмите кнопку Add.
  • Выберите Import From Clipboard.
  • Нажмите галочку в правом верхнем углу для сохранения.
  • Повторите те же действия для второй ссылки.

Шаг 4*. Исправление ошибок

Кажется, после обновления такие проблемы возникают не так часто, но бывает (на iOS и macOS). Поэтому если в какой-то моменте пропала возможность подключиться (выдает ошибку) к любому профилю конфигураци, то выполните следующие шаги для восстановления:

1. Откройте Karing.
2. Отключите соединение, нажав на кнопку внизу экрана (как обычно выключаете VPN).
3. Перейдите в раздел My Profiles.
4. Пролистайте список до профиля NaiveProxy v1 и выключите его с помощью тумблера.
5. Вернитесь в главное меню и выберите любой другой рабочий конфиг — теперь соединение должно установиться.
6. Спустя какое-то время (от 30 секунд) вы можете снова зайти в настройки и включить профиль NaiveProxy v1
